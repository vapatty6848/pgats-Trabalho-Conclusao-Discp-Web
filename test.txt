# Install the BrowserStack Cypress CLI
Passo a passo para integrar com o browserstack:

- npm install -g browserstack-cypress-cli
- browserstack-cypress init
- atualizar o username e access_key
- reduzir ao máximo a combinação de browsers (ideal é deixar 1 só)
- atualizar o run_settings.cypress_config_file para o caminho do arquivo de config do cypress
- adicionar o run_settings.cypress_version com a versão que quiser usar
- atualizar o run_settings.project_name para o nome do projeto de vocês
- atualizar o run_settings.parallels para 1
- adicionar dependências que foram usadas no projeto no run_settings.npm_dependencies
- browserstack-cypress run --sync --spec "atualizar com o caminho da spec"
